## 1. 项目概述

### 1.1 产品定位
JackDu ChatHub 是一款 Chrome 浏览器扩展，允许用户在**单一界面**中同时与多个 AI 聊天服务进行对话。通过统一的输入框，用户可以将消息**同步广播**到所有激活的 AI 窗口，实现高效的多模型对比和协作。

### 1.2 核心价值
- **效率提升**：一次输入，多平台响应
- **对比分析**：同时查看多个 AI 的回答
- **统一管理**：在一个界面管理所有 AI 对话，支持拖拽排序
- **个性化定制**：侧边栏、提示词库、自定义适配器全方位支持

### 1.3 目标用户
- AI 研究人员：对比不同模型的输出质量
- 内容创作者：获取多样化的创意灵感
- 开发者：测试 AI API 响应
- 普通用户：寻找最佳 AI 回答

## 3. 非功能需求

### 3.1 性能要求
- 消息广播延迟 < 100ms (广播指令)
- 总交互延迟 400-1000ms (针对反 Bot 机制)
- iframe 加载超时处理
- 内存占用优化（按需加载）

### 3.2 兼容性要求
- Chrome 88+
- Manifest V3

### 3.3 安全要求
- 同源消息验证
- 扩展 Origin 验证
- 无敏感数据存储

## 6. 附录

### 6.1 术语表
| 术语 | 定义 |
|------|------|
| Adapter | 服务适配器，定义 AI 服务的 URL 和 DOM 选择器 |
| ChatBot | 激活的 AI 实例，包含适配器信息和运行状态 |
| Content Script | 注入到 iframe 中的脚本，负责 DOM 操作 |
| Sync Mode | 同步模式，消息自动提交到 AI |
| Draft Mode | 草稿模式，消息仅填充不提交 |
